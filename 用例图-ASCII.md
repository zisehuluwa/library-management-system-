================================================================================
                              Include / Extend 关系
================================================================================

  借书(UC10)
      │
      ├──⟨include⟩→ 验证读者资格
      │                   ↓
      │              读者证有效？
      │               ↙         ↘
      │            有效         无效 → 拒绝借书
      │
      └──⟨include⟩→ 检查借阅规则
                      ↓
                 借阅数量未超？
                  ↙         ↘
               未超          已超 → 拒绝借书

  还书(UC11)
      │
      ├──⟨include⟩→ 检查超期状态
      │               ↓
      │          是否超期？
      │           ↙       ↘
      │         是         否
      │         ↓           ↓
      │    计算违约金    完成还书
      │
      ├──⟨extend⟩→ 损坏图书登记
      └──⟨extend⟩→ 遗失图书处理
